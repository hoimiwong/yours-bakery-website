# Bakery Website (Wix)

## Description
Developed a bakery website using Wix to showcase products, provide business information, and improve customer engagement through an online platform.

## Live Demo
https://tp077152.wixsite.com/yours-bakery

## Key Features
- Designed responsive website layout with navigation menu
- Displayed product listings and top-selling items
- Implemented contact form and membership registration
- Created gallery and feedback system for user interaction
- Developed promotion page to highlight offers

## Technologies Used
- Wix Website Builder
- Web Design (UI/UX)
- Content Management

## Project Purpose
This project demonstrates basic web development and digital marketing concepts, including user interface design and online business presentation.
